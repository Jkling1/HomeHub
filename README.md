# 🔥 IronMind Dashboard v2.0

A cutting-edge, gamified task management dashboard designed to forge discipline and unlock your potential through the four pillars of personal development.

![IronMind Dashboard](https://via.placeholder.com/800x400/1a1a1a/ff6b35?text=IronMind+Dashboard)

## 🎯 Features

### ⚡ Core Functionality
- **Four Pillars System**: Discipline (Flame), Growth (Mind), Endurance (Mountain), and Systems (Gear)
- **Dynamic Daily Tasks**: Rotating task pools with fresh challenges every day
- **XP & Leveling**: Earn experience points and advance through 10 distinct ranks
- **Streak Tracking**: Build momentum with daily completion streaks
- **Achievement System**: Unlock 10+ unique achievements with special rewards

### 🚀 Enhanced Features (v2.0)
- **🎨 Dual Theme Support**: Toggle between dark and light modes
- **💾 Local Storage Persistence**: Your progress is automatically saved
- **🔊 Audio Feedback**: Satisfying sound effects for task completion and level-ups
- **📊 Advanced Analytics**: Comprehensive statistics and progress tracking
- **🎯 Custom Tasks**: Create personalized tasks with custom XP rewards and icons
- **📅 Calendar View**: Visual monthly progress tracking for each quadrant
- **🏆 Special Challenges**: Participate in unique challenges for bonus rewards
- **📱 Mobile Responsive**: Optimized for all devices with touch-friendly interface
- **⌨️ Keyboard Shortcuts**: Power-user features for quick navigation
- **📈 Export/Import**: Backup and restore your data seamlessly

### 🎪 Advanced UI/UX
- **Smooth Animations**: Custom CSS animations and micro-interactions
- **Glass Morphism**: Modern translucent design elements
- **Gradient Effects**: Dynamic, eye-catching visual elements
- **Hover Enhancements**: Interactive feedback for all clickable elements
- **Progress Visualizations**: Animated progress bars and completion indicators
- **Notification System**: Elegant toast notifications for achievements and level-ups

## 🛠️ Technical Stack

- **React 18.2+** with modern hooks (useState, useEffect, useCallback, useMemo)
- **Lucide React** for beautiful, consistent icons
- **CSS3** with custom animations and transitions
- **Local Storage API** for data persistence
- **Web Audio API** for sound effects
- **Responsive Design** with mobile-first approach

## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/ironmind/dashboard.git
   cd dashboard
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` and enter the Forge!

## 📋 Usage

### 🎮 Getting Started
1. **Enter the Forge**: Complete the welcome tutorial
2. **Choose Your Path**: Select a quadrant to view available tasks
3. **Complete Tasks**: Click on tasks to mark them complete and earn XP
4. **Build Streaks**: Complete daily tasks to maintain your streak
5. **Unlock Achievements**: Reach milestones to earn special badges
6. **Level Up**: Advance through 10 ranks from Recruit to Transcendent

### ⌨️ Keyboard Shortcuts
- `1-4`: Quick navigate to quadrants (Flame, Mind, Mountain, Gear)
- `Space`: Mark selected task as complete
- `A`: Add new custom task
- `S`: Open statistics dashboard
- `T`: Toggle theme (Dark/Light)
- `E`: Export data
- `Esc`: Close modals/go back
- `?`: Show keyboard shortcut help

### 🎯 The Four Quadrants

#### 🔥 Discipline (Flame)
Focus on physical discipline and routine building:
- Morning workouts and physical challenges
- Building consistent wake-up routines
- Developing physical resilience

#### 🧠 Growth (Mind)
Dedicated to learning and mental development:
- Reading and continuous learning
- Skill development and knowledge acquisition
- Mental exercises and meditation

#### ⛰️ Endurance (Mountain)
Building courage and pushing personal boundaries:
- Difficult conversations and social challenges
- Stepping outside comfort zones
- Building mental toughness

#### ⚙️ Systems (Gear)
Organization and productivity systems:
- Planning and goal setting
- Workspace organization
- Process optimization and automation
